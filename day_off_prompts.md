# Day Off Prompts

자주 쓰는 Codex 프롬프트 모음입니다. 필요한 문장을 복사해서 Codex에 붙여 넣으면 됩니다.

## 일과 마무리

오늘 일과 마무리해줘.

오늘 한 일, 남은 일, 다음에 이어서 볼 포인트를 정리한 작업 로그를 만들어줘.
변경된 코드와 작업 로그를 함께 확인한 뒤, 적절한 커밋 메시지로 커밋하고 push 해줘.

IT 용어가 나오면 처음 한 번은 짧게 설명도 붙여줘.

## 작업 시작

오늘 작업을 시작하려고 해.

최근 작업 로그와 현재 변경 상태를 먼저 확인하고, 지금 이어서 할 만한 일을 요약해줘.
아직 코드는 바꾸지 말고, 먼저 상황 정리와 추천 다음 단계를 알려줘.

## 중간 정리

지금까지 진행한 내용을 중간 정리해줘.

이미 끝난 일, 아직 확인이 필요한 일, 다음으로 하면 좋은 일을 나눠서 설명해줘.
내가 다른 컴퓨터에서 이어서 봐도 이해할 수 있게 적어줘.

## 커밋 전 점검

커밋하기 전에 변경 내용을 점검해줘.

어떤 파일이 바뀌었는지, 실수로 들어간 내용은 없는지, 테스트나 확인이 필요한 부분은 없는지 확인해줘.
문제가 없으면 커밋 메시지도 추천해줘.

## 다른 컴퓨터에서 이어받기

이 작업을 다른 컴퓨터에서 이어서 할 수 있게 정리해줘.

현재 상태, 중요한 파일, 마지막으로 확인한 내용, 다음에 해야 할 일을 짧고 명확하게 적어줘.
필요하면 작업 로그 파일에도 반영해줘.
