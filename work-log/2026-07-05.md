## 용어 메모

- Commit: Git에서 변경 내용을 저장 지점으로 기록하는 것.
- Push: 내 컴퓨터의 commit을 GitHub 같은 원격 저장소로 올리는 것.
- Remote repository: 다른 컴퓨터에서도 받을 수 있는 온라인 저장소.
- Clone: GitHub 저장소 전체를 내 컴퓨터로 처음 내려받는 것.
- Pull: GitHub에 올라간 최신 변경사항을 내 컴퓨터로 받아오는 것.
- Google Drive: 문서, 이미지, PDF 같은 일반 파일을 보관하고 공유하기 좋은 저장소.
- GitHub: 코드와 작업 로그처럼 변경 이력을 남기며 관리해야 하는 파일에 적합한 저장소.
- MCP: Codex가 GitHub 같은 외부 서비스와 연결해 정보를 읽거나 작업할 수 있게 해주는 연결 방식.
